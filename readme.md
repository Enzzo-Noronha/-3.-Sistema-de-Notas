# 🗒️ Sistema de Notas

Aplicação para criar e gerenciar notas com persistência em banco de dados.

## 🚀 Tecnologias
- HTML, CSS, JavaScript
- Node.js
- Express
- SQLite3

## ⚙️ Funcionalidades
- Criar nota
- Listar notas
- Editar nota
- Excluir nota
